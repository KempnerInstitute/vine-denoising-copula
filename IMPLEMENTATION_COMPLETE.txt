KEY TECHNICAL IMPROVEMENTS
===========================

1. Correct ISE Integration on Non-Uniform Grids
   
   Formula: ISE = Σ (p_ij - q_ij)^2 * Δu_i * Δv_j
   
   - Uniform grid: Δu_i * Δv_j = constant → reduces to simple mean
   - Probit grid:  Δu_i * Δv_j varies → properly accounts for cell size
   
   Implementation in compute_density_losses():
     area = geometry['area']  # (1,1,m,m) tensor
     ise_tensor = (diff * area).view(B, -1).sum(-1)

2. Geometry Construction
   
   build_geometry(m, mode='uniform'|'probit', device):
     Returns dict with:
       - row_coords, col_coords: bin centers
       - row_widths, col_widths: bin widths Δu_i, Δv_j
       - area: per-cell area Δu_i * Δv_j
   
   Used throughout training for correct integration

3. Modular Architecture
   
   vdc/
   ├── train/           ← Training engine (new)
   ├── vine/            ← Inference interface
   ├── models/          ← Network architectures
   ├── data/            ← Data generation
   └── eval/            ← Evaluation utilities
   
   scripts/             ← Thin CLI wrappers only
