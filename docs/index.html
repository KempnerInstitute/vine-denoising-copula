Vine Denoising Copula

Amortized vine copula estimation, information decomposition, and pretrained model verification.

VDC trains a single bivariate edge estimator once on synthetic copulas and reuses it across all O(d²) vine edges, replacing expensive per-edge fitting with cheap GPU forward passes while preserving exact copula semantics, h-functions, and tractable likelihoods.

Method Overview

VDC amortizes bivariate copula estimation: a single pretrained U-Net maps empirical histograms to copula densities, IPFP projects to valid copulas, and the resulting edge operator is reused across all vine edges for density estimation, sampling, and information-theoretic decomposition.

Model Release Checklist

  1. Package the bundle from the canonical checkpoint.
  2. Upload the bundle to a Hugging Face model repo.
  3. Update the packaged manifest with the public repo id and revision.
  4. Run the verification script against the published model.

Key Assumptions

  • The pretrained model assumes continuous marginals and pseudo-observations in [0,1].
  • Vine fitting uses the simplifying assumption (constant conditional copulas).
  • The released checkpoint is frozen and versioned.
  • Default grid resolution is m=64; probit transform available for m=128.
